The Three Parameters

i
information density
e
entropy gradient
r
relation strength

These derive six rule categories: gravitational, tension, entropy, connection, memory, orbital.

Example

# Configure field
unified {
    i: 0.8
    e: 0.2
    r: 0.9
}

# Function
fn add a b {
    -> a + b
}

# Use
result = add(10, 20)
out "Done\n"
syscall.exit(0)

Self-Hosting

Each version includes a bootstrap compiler written in Wave itself:

# Compile the bootstrap (using Wave-C)
./wavec src/wavec_bootstrap.wave -o bootstrap

# Use bootstrap to compile
./bootstrap  # reads /tmp/input.wave, outputs /tmp/output

Quick Start

# Wave-C (recommended)
gcc -o wavec src/wave5.c -O2 -lm
./wavec hello.wave -o hello
./hello

# Wave-ASM
nasm -f elf64 src/wavec.asm -o wavec.o
ld wavec.o -o wavec

# Wave-BIN
nasm -f bin src/wavec.asm -o wavec.bin
