Quick Start

# Clone Wave-C (recommended)

cd wave-c

# Build compiler
gcc -o wavec src/wave5.c -O2 -lm

# Compile and run
./wavec examples/hello.wave -o hello
./hello

# Use self-hosting bootstrap
./wavec src/wavec_bootstrap.wave -o bootstrap
