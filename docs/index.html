Wave-C

Rule-Driven Compiler

Alpha Test 1.0

Core Idea

Wave doesn't hardcode compilation behavior. Everything is derived from three parameters.

Source
→
Unified Field
→
Derived Rules
→
Machine Code

The Three Parameters

i
information density
e
entropy gradient
r
relation strength

These three values derive six rule categories: gravitational, tension, entropy, connection, memory, orbital.

Example

# Configure the field
unified {
    i: 0.8
    e: 0.2
    r: 0.9
}

# Define a function
fn add a b {
    -> a + b
}

# Use it
result = add(10, 20)
out "Done\n"
syscall.exit(0)

Quick Start

# Build from source
gcc -o wavec src/wave5.c -O2 -lm

# Compile & run
./wavec hello.wave -o hello
./hello
